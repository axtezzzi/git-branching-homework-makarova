# Git Branching Homework

## Описание проекта

Консольное приложение, демонстрирующее работу с Git Branching.

## Использованные ветки

| Ветка | Что делал | Как попала в main |
|-------|-----------|-------------------|
| feature/profile | модуль профиля | Merge |
| feature/subjects | список дисциплин | Merge |
| feature/report | итоговый отчёт | Pull Request |
| experiment/broken-idea | тестовая идея | удалена |

## Коммиты

- merge pull reqest
- git add main.py
- add report module

- Add experimental idea

## Вывод

В ходе работы освоил создание веток, переключение между ними, локальное слияние, работу с Pull Request и удаление ненужных веток.
